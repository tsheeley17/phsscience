Peebles High School Physics Calculator Working

Welcome to the PHS Physics Calculator! We created this site in hopes of creating a tool that allows you to do all of your physics-based calculations. This calculator can calculate a lot of the more common formulas found in physics, as well as providing several useful links to sites regarding information about PHS and other physics/science related sites. We also included a link to the original Python code that this calulator was based on.


This calculator was originally meant to help practice learning the Python coding language, as well as practice skills in physics. It wasn't until we realized how useful of a tool this could actually be that we decided to put it on the Internet for anyone to use!
